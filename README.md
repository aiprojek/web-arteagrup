# Artea Grup Linktree

Selamat datang di repositori resmi untuk halaman Linktree Artea Grup. Proyek ini adalah halaman arahan (landing page) yang modern dan responsif, dirancang untuk menjadi pusat dari semua tautan penting yang terkait dengan merek minuman Artea Grup.

![Screenshot Aplikasi Artea Grup](https://firebasestorage.googleapis.com/v0/b/ai-marketplace-bad91.appspot.com/o/artea-grup-screenshot.png?alt=media&token=12140409-299f-4318-9125-9fc2174c8612)

## Deskripsi

Halaman ini berfungsi sebagai "pohon tautan" (linktree) yang indah dan fungsional untuk merek minuman Artea Grup, yang menaungi **Artea** dan **Janji Koffee**. Halaman ini menyediakan akses cepat dan mudah ke menu, informasi lokasi, kontak, media sosial, dan fitur unik lainnya dalam satu antarmuka yang elegan dan mudah dinavigasi.

## Fitur Utama

- **Desain Responsif:** Tampilan yang dioptimalkan untuk semua perangkat, dari ponsel hingga desktop.
- **Panel Geser & Modal Dinamis:** Pengalaman pengguna yang mulus saat menavigasi ke halaman detail seperti menu atau informasi "Tentang Kami". Aplikasi secara cerdas beralih antara panel geser (mobile) dan modal layar penuh (tablet/desktop).
- **Pusat Informasi:** Akses terpusat ke menu minuman untuk Artea dan Janji Koffee, lengkap dengan varian rasa.
- **Integrasi Peta:** Tautan langsung ke Google Maps untuk setiap lokasi gerai, memudahkan pelanggan untuk menemukan kami.
- **Rekomendasi Minuman Berbasis AI:** Fitur inovatif yang menggunakan Gemini API untuk memberikan rekomendasi minuman yang dipersonalisasi berdasarkan keinginan pengguna.
- **Kontak & Media Sosial:** Tautan cepat untuk menghubungi kami melalui WhatsApp dan Email, serta untuk mengikuti kami di platform media sosial.
- **Fungsionalitas Bagikan/Salin:** Tombol praktis untuk membagikan halaman atau menyalin URL ke clipboard.
- **Dibangun dengan Teknologi Modern:** Menggunakan React, TypeScript, dan Tailwind CSS untuk antarmuka yang cepat dan modern.

## Menjalankan Proyek Secara Lokal

Proyek ini adalah aplikasi web statis. Untuk menjalankannya secara lokal, Anda hanya perlu:

1.  Mengkloning repositori ini.
2.  Membuka file `index.html` di browser web pilihan Anda.

Tidak ada proses build atau dependensi server yang rumit yang diperlukan untuk menjalankan antarmuka pengguna.
